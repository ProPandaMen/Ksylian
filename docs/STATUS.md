# Статус разработки Ksylian

Обновлено: 2026-07-18

## Текущий фокус

Активный этап: Этап 1, стабилизация и безопасность.

Ближайшие приоритеты:

- Усилить безопасность host agent.
- Перестать запускать управляемые Minecraft-серверы от root.
- Вынести долгие операции из синхронных HTTP-запросов.
- Сделать публичный домен и proxy настраиваемыми из панели.

## Уже сделано

- Пользователи перенесены из `users.json` в SQLite.
- Обновлен дизайн страниц авторизации, настроек, пользователей, мониторинга, серверов, страницы сервера и создания сервера.
- Цветовые темы вынесены в отдельные файлы.
- Добавлен базовый путь создания Fabric и Forge серверов на уровне agent.
- Paper и Purpur временно убраны из активного интерфейса, но остаются в roadmap как будущие полноценные загрузчики.
- Добавлен базовый Minecraft proxy Ksylian.
- Подготовлена маршрутизация через wildcard-домен `<server>.<domain>`.
- Публичные адреса серверов показываются через agent при настроенном `KSYLIAN_PUBLIC_DOMAIN`.
- Онлайн и максимальное число игроков читаются через Minecraft status ping.

## Примечания по roadmap

Основной roadmap хранится в `docs/ROADMAP.md`.

Roadmap считаем источником правды. Если текущая реализация временно отличается от roadmap, это фиксируем в статусе как технический долг, а не как изменение направления.

## Следующая кандидатная задача

Безопасность agent:

- Сделать `KSYLIAN_AGENT_TOKEN` обязательным.
- Запрещать запуск agent без токена.
- Ограничить доступ к agent только backend-сервисом или локальной сетью.
- Добавить журнал действий agent для создания, запуска, остановки, удаления серверов и изменений proxy.
